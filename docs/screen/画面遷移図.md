# 画面遷移図

## 概要
タスク管理アプリケーションの画面遷移を定義する。

## 全体構成

```
+-------------------+       +-------------------+
|   タスク一覧      |       | カテゴリー一覧    |
| (SCR-TASK-001)    |       | (SCR-CAT-001)     |
+-------------------+       +-------------------+
         |                           |
         +---------------------------+
```

## タスク管理機能

```
                    +---------------------+
                    |     タスク一覧      |
                    |   (SCR-TASK-001)    |
                    +----------+----------+
                               |
         +---------------------+---------------------+
         |                     |                     |
         v                     v                     v
+---------------------+ +---------------------+ +---------------------+
|     タスク登録      | |     タスク詳細      | |     タスク編集      |
|   (SCR-TASK-002)    | |   (SCR-TASK-003)    | |   (SCR-TASK-004)    |
+----------+----------+ +----------+----------+ +----------+----------+
           |                       |                       |
           |                       +-----------------------+
           |                       |                       |
           +-----------------------+-----------------------+
                                   |
                                   v
                    +---------------------+
                    |     タスク一覧      |
                    |   (SCR-TASK-001)    |
                    +---------------------+
```

| No | 遷移元 | 遷移先 | トリガー | 備考 |

| 1 | SCR-TASK-001 | SCR-TASK-002 | 「新規タスク」ボタン押下 | - |
| 2 | SCR-TASK-001 | SCR-TASK-003 | タスクタイトルクリック | - |
| 3 | SCR-TASK-001 | SCR-TASK-004 | 編集ボタン押下 | - |
| 4 | SCR-TASK-002 | SCR-TASK-001 | 登録成功 / キャンセル | - |
| 5 | SCR-TASK-003 | SCR-TASK-001 | 「一覧に戻る」リンク押下 | - |
| 6 | SCR-TASK-003 | SCR-TASK-004 | 「編集」ボタン押下 | - |
| 7 | SCR-TASK-003 | SCR-TASK-001 | 削除成功 | 確認モーダル経由 |
| 8 | SCR-TASK-004 | SCR-TASK-003 | 更新成功 / キャンセル | - |
