# 画面一覧

## 概要
タスク管理アプリケーションの画面一覧を定義する。

## 画面ID命名規則
| フォーマット | 説明 | 例 |
|:---|:---|:---|
| `SCR-{機能}-{連番}` | 機能別画面 | SCR-TASK-001, SCR-CAT-001 |

### 機能コード一覧
| 機能コード | 機能名 | 説明 |
|:---|:---|:---|
| TASK | タスク管理 | タスクのCRUD操作 |
| CAT | カテゴリー管理 | カテゴリーのCRUD操作 |
| AUTH | 認証 | ログイン・ログアウト |
| CMN | 共通 | 共通機能・エラー画面等 |

## 画面詳細

### SCR-TASK-001: タスク一覧
- **目的**: 登録されているタスクを一覧表示し、検索・フィルタリングを行う
- **主な機能**:
  - タスク一覧のテーブル表示
  - ステータス/優先度/カテゴリーによるフィルタリング
  - 新規登録画面への遷移
  - 詳細画面/編集画面への遷移
- **モックアップ**: `docs/mockups/01_task_list.html`
- **設計書**: `docs/screen/画面設計書/SCR-TASK-001_タスク一覧.md`

### SCR-TASK-002: タスク登録
- **目的**: 新規タスクを登録する
- **主な機能**:
  - タスク情報の入力フォーム
  - 入力バリデーション
  - 登録処理
  - キャンセル（一覧へ戻る）
- **モックアップ**: `docs/mockups/02_task_new.html`
- **設計書**: `docs/screen/画面設計書/SCR-TASK-002_タスク登録.md`

### SCR-TASK-003: タスク詳細
- **目的**: 選択したタスクの詳細情報を表示する
- **主な機能**:
  - タスク情報の詳細表示
  - 編集画面への遷移
  - タスク削除（確認モーダル付き）
  - 一覧画面への戻る
- **モックアップ**: `docs/mockups/03_task_detail.html`
- **設計書**: `docs/screen/画面設計書/SCR-TASK-003_タスク詳細.md`

### SCR-TASK-004: タスク編集
- **目的**: 既存タスクの情報を編集する
- **主な機能**:
  - 既存タスク情報の表示・編集
  - 入力バリデーション
  - 更新処理
  - キャンセル（詳細へ戻る）
- **モックアップ**: `docs/mockups/04_task_edit.html`
- **設計書**: `docs/screen/画面設計書/SCR-TASK-004_タスク編集.md`

### SCR-CAT-001: カテゴリー一覧
- **目的**: 登録されているカテゴリーを一覧表示し、カテゴリーの管理を行う
- **主な機能**:
  - カテゴリー一覧のテーブル表示（表示順でソート）
  - カテゴリー名、説明、カラー、表示順の表示
  - 新規登録画面への遷移
  - 編集画面への遷移
  - カテゴリー削除（確認モーダル付き）
- **モックアップ**: `docs/mockups/07_category_list.html`
- **設計書**: `docs/screen/画面設計書/SCR-CAT-001_カテゴリー一覧.md`

### SCR-CAT-002: カテゴリー登録
- **目的**: 新規カテゴリーを登録する
- **主な機能**:
  - カテゴリー情報の入力フォーム（名前、説明、カラー、表示順）
  - カラーピッカーによる色選択
  - 入力バリデーション
  - 登録処理
  - キャンセル（一覧へ戻る）
- **モックアップ**: `docs/mockups/08_category_form.html`
- **設計書**: `docs/screen/画面設計書/SCR-CAT-002_カテゴリー新規登録.md`

### SCR-CAT-003: カテゴリー編集
- **目的**: 既存カテゴリーの情報を編集する
- **主な機能**:
  - 既存カテゴリー情報の表示・編集
  - カラーピッカーによる色変更
  - 入力バリデーション
  - 更新処理
  - キャンセル（一覧へ戻る）
- **モックアップ**: `docs/mockups/09_category_edit.html`
- **設計書**: `docs/screen/画面設計書/SCR-CAT-003_カテゴリー編集.md`

### SCR-CMN-001: 404エラー
- **目的**: 存在しないリソースへのアクセス時にエラーを表示
- **主な機能**:
  - エラーメッセージ表示
  - 一覧画面への遷移リンク
- **モックアップ**: `docs/mockups/05_error_404.html`
- **設計書**: `docs/screen/画面設計書/SCR-CMN-001_404エラー.md`

### SCR-CMN-002: 500エラー
- **目的**: システムエラー発生時にエラーを表示
- **主な機能**:
  - エラーメッセージ表示
  - 一覧画面への遷移リンク
- **モックアップ**: `docs/mockups/06_error_500.html`
- **設計書**: `docs/screen/画面設計書/SCR-CMN-002_500エラー.md`
